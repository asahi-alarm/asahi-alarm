Asahi Arch Linux ARM

Asahi Alarm is an Arch Linux ARM distribution for Apple Silicon macs. We build on top of the Asahi Linux project, without which this would not be possible.

We offer KDE plasma, GNOME, XFCE, MATE, LXQt and Hyprland desktop environments, as well as a minimal installation (do read the manual install steps if you choose this option).

Install

Run the following from macOS Terminal to install Asahi Alarm

curl https://asahi-alarm.org/installer-bootstrap.sh | sh

Choose minimal for a bare bones install, suitable for a server. Choose desktop if you want to intall a desktop environment. At first boot, the calamares installer will ask which desktop you want, and install all required packages. There is now also a BTRFS variant to minimal and desktop. It's setup with 2 subvolumes @ for / and @home for /home.

Steam

sudo pacman -Sy steam
